# Tetris & Snake Dual Game

This project contains two classic games linked together: **Tetris** and **Snake**.

## How to Play
1. The game starts with **Tetris**.
2. Play as long as you can to accumulate points.
3. When you lose in Tetris, the game **automatically transitions** to **Snake**.
4. Try to survive in Snake to add to your total score!

## Controls
- **Arrow Keys**: Move & Rotate (Tetris) / Steer (Snake)
- **Space**: Hard Drop (Tetris)
- **P**: Pause
